Features

Member-Exclusive Login with Google SSO

The login functionality operates efficiently and includes Google authentication to facilitate secure sign-in for users listed in the database's user table.

Registration is intentionally disabled to maintain organizational security and control access.

Comprehensive Interface

The interface provides detailed profiles of tenants, owners, and locations, facilitating essential operations such as editing, deleting, and updating.

Moreover, it serves as a repository for crucial documents like lease agreements.

Multi-Platform Accessibility

Accessible seamlessly across various platforms and screen sizes, ensuring convenience and flexibility for users.

Data Analytics

Interactive pie charts and statistical tools are integrated into the website, empowering users to analyze property data comprehensively.
